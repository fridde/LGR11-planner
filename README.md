# LGR-planner

The following manual will be in Swedish. If you are interested in contributing or in rebuilding this project but can't read Swedish, contact me instead.

## Det viktigaste
**LGR11-planner** är ett webbaserat verktyg för att förenkla pedogiska planeringar enligt LGR11.

Den aktuella versionen av planeraren finns på [sigtunanaturskola.se/lgr11](http://www.sigtunanaturskola.se/lgr11/) och projektets manual samt källkoden finns på [GitHub](https://github.com/fridde/LGR11-planner). Planeraren är fortfarande under utveckling.

## Introduktion

Skolverket utfärdarde *Läroplanen för grundskolan, förskoleklassen och fritidshemmet 2011* för många år sedan, men än så länge saknas det ett **verktyg** för att på snabbt och översiktligt sätt skapa pedagogiska planeringar utan att använda "copy-paste-metoden". 

Ofta öppnar man [läroplanen i pdf-form](http://www.skolverket.se/publikationer?id=2575) och kopierar (eller värre: använder papper och penna) text till sin mall för pedagogiska planeringen. Krångligt.
